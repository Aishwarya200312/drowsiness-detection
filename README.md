# 🚗 Drowsiness Detection System

## 📌 Project Overview

This project is a real-time **Drowsiness Detection System** developed using Python. It detects driver fatigue by monitoring eye closure and yawning using computer vision techniques and alerts the user to prevent accidents.

---

## 🎯 Features

* 👁️ Eye closure detection using Eye Aspect Ratio (EAR)
* 😮 Yawning detection using Mouth Aspect Ratio (MAR)
* 🔔 Real-time alert system (visual + audio)
* 📷 Live webcam-based detection
* ⚡ Fast and efficient performance

---

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe

---

## 📂 Project Files

* `main.py` – Main execution file
* `drowsiness.py` – Core detection logic
* `face_mesh.py` – Facial landmark detection
* `test_camera.py` – Camera testing
* `alarm.wav` – Alert sound

---

## ▶️ How to Run the Project

1. Clone the repository
2. Install required libraries:

   ```bash
   pip install opencv-python mediapipe numpy
   ```
3. Run the project:

   ```bash
   python main.py
   ```
